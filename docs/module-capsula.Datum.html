Class: Datum

capsula.Datum(opt_dataopt)

Datum is a specific public property of a capsule. It serves to store the given information and to protect them from illegal access.

getFQName(opt_sepopt) → {string}

Returns the fully qualified name of this datum, using the given separator if provided (if not, the :: is used by default).

The fully qualified name comprises the name of this datum, the name of the owner capsule of this datum, the name of its owner, and so on all the way up the capsule hierarchy.
