Class: Input

capsula.Input(opt_nameopt, opt_functionopt)

  • Operation
    • Input

Input operation is a specific public property of a capsule. Input operations go along with output operations; they are complementary concepts.

Constructor

new Input(opt_nameopt, opt_functionopt)

Creates new input operation as a property of the capsule that represents the current context of execution, with the given name and implementation function (if provided). The given name can later be used to obtain a reference to this operation from its owner capsule using getInput or getOperation.
Parameters:
Name Type Attributes Description
opt_name string <optional>
 optional name of input operation to create
opt_function function <optional>
 optional function that gets called every time input operation gets called, i.e. it is a sort of implementation of the input operation to be created
